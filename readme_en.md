# What does this program do?
---
This program is capable of processing inputable matrixes.  
The program can summ, substract, process input matrixes, outputting results in console/output file.
# User guide:  
* execute the exe file on your PC
* choose the format (press the button) + Enter)
* follow instructions of the program.
---
__Note__:
the format of input-file **MUST** be like that:
```
1,2,3
4,5,6
7,8,9
+
9,8,7
6,5,4
3,2,1
```

Don't forget to save the file after formatting, also.  
input/output files are saved and stored in main folder.
